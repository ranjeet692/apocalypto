Complex Number Problem Solution

Create a class called Imaginary Number with the following attributes and behaviour.
						
As we know any Imaginary number will have real part as well as imaginary part, 
so any imaginary object will have two attributes corresponding to real and imaginary parts.
 
Object Attributes:
private double realPart
private double Imaginary Part.
 
Provide methods which will facilitate Modulous/Addition/Sub
traction/Multiplication of two
imaginary numbers.

Description:
Program will take 5 integers(p,q,r,s,t) as input.
Numbers will be p+iq and r+is.
t is the function which should be executed.
t=1,2,3,4 for Sum of Modulous of numbers,Addition,Subtraction,Muliplication respectively.
Sample Input ::
2 3
4 5
2
Sample Output ::
6+8i

Limits
p,q,r,s can be any positive or negative number.
1<=t<=4

In Subtraction second number has to be reduced from first
